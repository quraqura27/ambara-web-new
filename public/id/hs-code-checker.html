Tanya Jawab Seputar Lartas

Apa itu Lartas?

Lartas (Larangan dan Pembatasan) adalah regulasi tata niaga dari Kementerian terkait di Indonesia. Jika HS Code Anda terkena Lartas, Anda wajib mengurus izin persetujuan impor (seperti SNI atau PI) sebelum kargo tiba. Jika gagal, kargo akan ditahan di Jalur Merah.

Mengapa saya butuh pengecekan via PPJK?

Prefix Bab HS Code bisa memprediksi risiko secara akurat, tapi database resmi INSW/CEISA kadang memblokir HS Code 8-digit yang sangat spesifik. PPJK berlisensi resmi (seperti Ambara) menggunakan jaringan Edifact mandiri untuk menarik data real-time, memastikan izin mana yang wajib dipenuhi hari ini.
