Frequently Asked Questions

What is Lartas?

Lartas (Larangan dan Pembatasan) refers to Prohibitions and Restrictions established by Indonesian Ministries. If your HS Code triggers Lartas, you must secure exact permits (like Persetujuan Impor or SNI) before the cargo arrives, otherwise it faces Red Lane (Jalur Merah) rejection.

Why do I need a PPJK lookup?

While HS Code Chapter prefixes can accurately predict risk, the official INSW/CEISA database dynamically blocks specific 8-digit codes. Licensed PPJKs (Customs Brokers) like Ambara use encrypted Edifact gateways to retrieve the exact, legally-binding permit requirements for your shipment today.
